deepin搭建java开发环境

发表于 2021-04-11 19:20:49 更新于 2021-05-24 16:11:40 分类于 linux ， deepin 阅读次数：
本文字数： 2k 阅读时长 ≈ 2 分钟

安装jdk

  安装之前最好先获取管理员权限。

# 获取后需要输入密码
sudo su

下载jdk

  进入指定目录，下载jdk文件，我自己通常喜欢下载在/usr/local/src/jdk目录中，jdk16。
  /usr/bin是系统的软件存放的位置。
  /usr/local/bin是我们自己的软件存放的位置。

cd /usr/local/src/jdk

wget https://download.oracle.com/otn-pub/java/jdk/16+36/7863447f0ab643c585b9bdebf67c69db/jdk-16_linux-x64_bin.tar.gz?AuthParam=1618140718_ccecc5f8b3763f1feed74a49a7f7cbb8

将下载的jdk解压缩

  通过wget下载的压缩包文件名称会奇奇怪怪的。如果通过浏览器下载则没有该问题。

# 重命名文件
mv 'jdk-16_linux-x64_bin.tar.gz?AuthParam=1618140718_ccecc5f8b3763f1feed74a49a7f7cbb8' jdk-16_linux-x64_bin.tar.gz

# 解压缩文件
tar xvf jdk-16_linux-x64_bin.tar.gz

重命名文件夹（可选）

  对解压后的文件夹重命名，也可以不做。

mv jdk-16_linux-x64_bin jdk16

配置环境变量

  jdk需要配置PATH和CLASSPATH环境变量。

# 系统的环境变量写在/etc/profile文件中
vi /etc/profile

# 在编辑器末尾添加对应的环境变量
# 这里的路径结合自己的实际安装路径
JAVA_HOME=/usr/local/jdk/jdk16
CLASSPATH=$JAVA_HOME/lib
export CLASSPATH
export PATH=$PATH:&JAVA_HOME/bin

# 退出vi并保存，这里需要强制保存
esc
:wq!

# 刷新使环境变量生效
source /etc/profile

# 验证，查看版本号
java -version

安装maven

下载maven

  将maven下载进指定目录，这里依照个人习惯，我放在/usr/local/src/maven。maven3.8.1

cd /usr/local/src/maven

wget https://mirrors.bfsu.edu.cn/apache/maven/maven-3/3.8.1/binaries/apache-maven-3.8.1-bin.tar.gz

解压缩文件

  对下载的maven文件解压缩。

tar xvf apache-maven-3.8.1-bin.tar.gz

重命名文件夹

  这一步可选操作，目的是为了文件目录整洁。也可以不做。

mv apache-maven-3.8.1-bin maven3

配置环境变量

  maven也需要配置PATH环境变量。

vi /etc/profile

# 在编辑器末尾添加对应的环境变量
# 这里的路径结合自己的实际安装路径
MAVEN_HOME=/usr/local/maven/maven3
export PATH=$PATH:&MAVEN_HOME/bin

# 退出vi并保存，这里需要强制保存
esc
:wq!

# 刷新使环境变量生效
source /etc/profile

# 验证，查看版本号
mvn -v

安装tomcat

下载tomcat

  将tomcat下载进指定的位置，我放在/usr/local/src/tomcat。tomcat10

cd /usr/local/src/tomcat

wget https://mirrors.tuna.tsinghua.edu.cn/apache/tomcat/tomcat-10/v10.0.5/bin/apache-tomcat-10.0.5.tar.gz

解压压缩包

  解压下载的压缩文件。

tar xvf apache-tomcat-10.0.5.tar.gz

重命名文件夹

  这一步可选操作，目的是为了文件目录整洁。也可以不做。

mv apache-tomcat-10.0.5 tomcat10

启动

  进入tomcat的安装目录就可以启动tomcat了。

# 进入tomcat的安装位置
cd /usr/local/src/tomcat/tomcat10/bin

# 启动tomcat
startup.sh

  启动完成后可以在localhost:8080查看是否启动成功。
